# ExchangeRateCalculator
환율 계산기 APP


- 현재 환율 정보를 실시간으로 불러와 계산합니다.
- 송금가능액은 0 ~ 10,000 USD
- 송금 국가는 미국(USD)으로 고정, 수취 국가는 한국(KRW), 일본(JPY), 필리핀(PHP)
- 수취 국가를 변경하면 환율 및 수취금액도 자동으로 변경됩니다.
